## 矩阵计算
以fx-cn991为例，

先按矩阵，可以自定义MatA，B，C，D四个矩阵（最高4x4）
- 设置好数字，按AC确认
- 按**OPTN**键选择矩阵运算
- $x^{-1}$ 进行矩阵求逆
- OPTN中还有转置矩阵、行列式等运算

[CASIO卡西欧991如何求逆矩阵-百度经验](https://jingyan.baidu.com/article/380abd0aa0a5c95d90192cba.html)
