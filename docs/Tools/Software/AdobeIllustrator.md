## 形状

### 形状裁剪

1. 打开AI软件，新建画布，将需要裁剪的图片拉进AI中。
2. 在左边的工具栏中，长按矩形工具选择椭圆工具。
3. 使用椭圆工具，在图片上面按住shift键绘制一个正圆形。
4. 同时选中图片和圆形，单击鼠标右键点击建立剪切蒙版。
5. 图片就被裁剪成圆形了。
