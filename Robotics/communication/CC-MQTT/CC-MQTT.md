### Docker环境
[下载 EMQX 开源版](https://www.emqx.com/zh/downloads-and-install/broker?os=Docker)
```shell title="安装EMQX"
docker pull emqx/emqx:5.8.3
```

```shell title="启动EMQX"
docker run -d --name emqx -p 1883:1883 -p 8083:8083 -p 8084:8084 -p 8883:8883 -p 18083:18083 emqx/emqx:5.8.3
```
