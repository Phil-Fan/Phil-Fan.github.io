## Quick Start

```bash title="Clone the repository"
git clone https://github.com/Phil-Fan/Phil-Fan.github.io.git
```

```bash title="Install dependencies"
cd Phil-Fan.github.io
pip install -r requirements.txt
```


```bash title="Serve the website"
mkdocs serve
```
